
Machine Learning: Prediction and Classification Models

This project analyzes regression and classification problems using common machine learning datasets, with the goal of understanding how different models behave and perform. 

Objectives
Analyze and compare different machine learning models  
Understand the effect of model complexity on performance  
Evaluate models using appropriate metrics  
Examine challenges in high dimensional data  
Design and train a neural network model  

Key Insights
Model complexity significantly affects performance and generalization  
Classical models struggle with high-dimensional image data  
Proper hyperparameter selection improves model performance  
Neural networks achieve better performance when properly tuned  

Tools and Technologies
Python  
Pandas  
NumPy  
Matplotlib  
Scikit-learn  
PyTorch  

Project Structure
regression_modeling.ipynb – regression analysis and modeling 
classification_models.ipynb – classical classification models 
neural_network_classification_pytorch.ipynb – neural network classification 
neural_network_regression_pytorch.ipynb – neural network regression 
final_report.pdf – project report
